# Runs Folder

This folder is where real datasets live.

**No fabricated datasets are included in this repository.**
A template structure is provided so every run is stored consistently and auditably.

## Required Structure (Per Run)
- `runs/<run_id>/`
  - `run_manifest.json` (required)
  - `raw/` (required)
  - `plots/` (recommended)
  - `notes/` (optional)

## Template Run
A `template_run/` is provided as a scaffold only.
It contains no measurements.
