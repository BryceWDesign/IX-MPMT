# Assembly Checklist (H6-C12)

## Mechanical
- [ ] Base plate mounted and marked (+X / Pod1 direction)
- [ ] Core mount installed with orientation index
- [ ] Pod mounts installed (1..6) with hard stops
- [ ] Core capsule mounted, indexed, cable exit correct
- [ ] All fasteners torqued per spec and torque-striped

## Sensor Skin (C12)
- [ ] Node mounts installed and labeled (A1..A6, B1..B6)
- [ ] Sensors mounted rigidly; wiring strain-relieved
- [ ] Stand accelerometer (S-A) mounted rigidly and labeled
- [ ] Rig accelerometer (R-A) mounted rigidly and labeled

## Power Spine
- [ ] Current limiting present and verified
- [ ] Dump/bleeder physically installed and verified
- [ ] Touch-safe test points labeled
- [ ] Interlock loop wired and tested

## Harness
- [ ] Fixed routing path installed with clamps/guides
- [ ] Power harness separated from sensor harness where possible
- [ ] No free-hanging loops near measurement axis
- [ ] Strain relief at each pod and at trunk

## Bring-up
- [ ] Sensors read plausible baselines (no saturation)
- [ ] `sync_out` captured by DAQ
- [ ] Toggle test into dummy loads recorded successfully
- [ ] Discharge verified by meter before touching
